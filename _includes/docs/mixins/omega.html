Omega View source

The omega() mixin can be called on any element to remove its gutter, regradless of its position in the grid hierarchy or display property. It explicitly designates an element as the last of its row. It can target a specific element, or every nth-child occurance. It works with both block | inline-block and table layouts.

Syntax

{% highlight scss %} @include omega($target $display, $direction); {% endhighlight %}

Optional Arguments

  • $target string : The pseudo selector to target.
    • Values > any valid nth-child value (should be single-quoted if contains operands) | auto to target the :last-child pseudo selector.
    • Default > null
  • $display string : The display property of the calling element. Useful when working with table layouts.
    • Values > block | inline-block | table
    • Default > block
  • $direction string : The direction of the layout. Useful when working with right-to-left layouts.
    • Values > right | left
    • Default > right

Example

{% highlight scss %} .element { @include omega; } .table-cell { @include omega('3n + 1' table, left); } // CSS .element { margin-right: 0; } .table-cell:nth-child(3n + 1) { padding-left: 0; } {% endhighlight %}
